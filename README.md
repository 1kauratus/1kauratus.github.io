Welcome to the Realtor Website Internship Project, a modern, immersive real estate platform designed and developed as part of my internship at FUTURA. This project blends elegant UI/UX design with advanced web technologies to provide users with a futuristic way to browse global property listings.

🌍 Live the World, One Property at a Time
The centerpiece of the site is a 3D interactive globe, giving users an intuitive and visually stunning way to explore properties worldwide. Instead of scrolling through endless listings, users can rotate the Earth, zoom in on regions, and click on markers to discover properties in real-time.

🚀 Features
🌐 3D Rotating Globe Interface
Explore properties visually across the globe with clickable location markers.

🏠 Dynamic Property Listings
Each marker reveals detailed property information, including images, descriptions, and contact options.

📱 Fully Responsive Design
Optimized for desktop, tablet, and mobile devices.

✨ Smooth Animations with GSAP
Enhanced user experience with high-quality scroll and hover animations using GSAP and ScrollTrigger.

🗺️ Interactive Mapping with Leaflet.js
Lightweight mapping library to handle globe interactivity and property geolocation.

🔍 Filtered Search Interface (Coming Soon)
Sort listings by location, price, type, and more.
